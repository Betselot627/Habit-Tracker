# 🚀 Quick Start - Habit Tracker

## ✅ Clerk Removed - Back to Original

Your app now uses the original JWT authentication system.

## Start the Application

### Terminal 1 - Backend

```bash
cd server
npm run dev
```

### Terminal 2 - Frontend

```bash
cd client
npm run dev
```

## Open Browser

Navigate to: `http://localhost:5173`

## First Time Setup

1. **Register Account**
   - Click "Create Account"
   - Enter name, email, password
   - Click "Create Account"

2. **Login**
   - Enter email and password
   - Click "Sign In"

3. **Start Using**
   - Create your first habit
   - Mark it as complete
   - Track your progress!

## Features

- ✅ Register/Login with email & password
- ✅ Create habits with categories
- ✅ Daily, weekly, yearly, or custom frequency
- ✅ Mark habits as complete
- ✅ View statistics and progress
- ✅ Dark mode support
- ✅ Mobile responsive

## That's It!

No external services needed. Everything runs locally.

Enjoy tracking your habits! 🎉
